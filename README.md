# 📊 SQL Sales Data Analysis Project

## 🔍 Project Overview

This project focuses on cleaning, analyzing, and extracting business insights from a sales dataset using SQL.

The project demonstrates practical SQL skills including:

* Data cleaning
* Exploratory data analysis
* Business problem solving
* Aggregations and filtering
* Query optimization concepts

---

## 📁 Dataset

The dataset contains:

* Customer information
* Product details
* Regional sales
* Order dates
* Revenue data

---

## 🧹 Data Cleaning Tasks

Performed using SQL:

* Removed duplicate records
* Handled NULL values
* Corrected inconsistent formats
* Validated numeric values
* Standardized text entries

---

## 📊 Exploratory Data Analysis

Key analysis performed:

* Total sales by region
* Top-selling products
* Monthly sales trends
* Average customer spending
* Revenue distribution

---

## 💡 Key Insights

* The West region generated the highest revenue
* Laptop products contributed most to sales
* Sales increased consistently over time

---
